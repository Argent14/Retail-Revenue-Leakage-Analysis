# Retail Revenue Leakage & Customer Profitability Analysis

## Project Overview

Businesses often experience revenue loss due to uncontrolled discounts, product returns, and unprofitable customers. This project explores how data analytics can be used to identify revenue leakage and improve profitability using an interactive Power BI dashboard.

The dashboard analyzes retail sales data to uncover patterns in revenue performance, profit margins, customer value, and operational inefficiencies.

This project demonstrates my ability to transform raw business data into actionable insights using Power BI and data modeling techniques.

---

## Business Problem

Many organizations focus on revenue growth without fully understanding where profit is being lost.

Key questions addressed in this analysis:

• Which product categories generate high revenue but low profit?
• How much revenue is lost through discounting strategies?
• Which products consistently generate negative profit?
• Which customers contribute the most to profitability?
• What operational factors are driving return rates?

---

## Dataset Description

The dataset contains retail transaction records with attributes including:

• Product ID
• Customer ID
• Quantity sold
• Unit price
• Discount percentage
• Return indicator
• Product category
• Region and transaction date

These variables allow for the analysis of both **sales performance and profitability drivers**.

---

## Analytical Approach

The analysis followed a structured business intelligence workflow:

1. Data Preparation
   Data was cleaned and transformed within Power BI to ensure consistency and accuracy.

2. Data Modeling
   Custom DAX measures were created to calculate revenue, profit, profit margins, return rates, and discount impact.

3. Exploratory Analysis
   Interactive visualizations were developed to identify trends across products, regions, and customers.

4. Business Insight Generation
   Patterns in discount behavior, return rates, and customer profitability were analyzed to identify revenue leakage.

5. Strategic Recommendations
   Insights were translated into practical recommendations for improving pricing strategy and operational efficiency.

## Key Insights

• High discount levels significantly reduce profit margins in certain product categories.
• A small number of products consistently generate negative profit.
• Some high-revenue customers produce low profitability due to excessive discounts.
• Return rates in certain categories create additional margin pressure.

---

## Strategic Recommendations

• Introduce controlled discount policies to protect margins.
• Review and potentially discontinue consistently unprofitable products.
• Implement customer profitability segmentation for better pricing strategies.
• Improve quality control for categories with high return rates.
